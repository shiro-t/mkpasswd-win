
usage: mkpasswd [args] [num]
  where arguments are:
    -l #      (length of password, default = 8 )
    -d #      (min # of digits, default = 1 )
    -c #      (min # of lowercase chars, default = 3 )
    -C #      (min # of uppercase chars, default = 2 )
    -s #      (min # of special chars, default = 1 )
   num        repeat num times
